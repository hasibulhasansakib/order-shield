# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.6] - 2026-07-08
### Fixed
- Stopped the GitHub Updater from accidentally hijacking other plugin updates.

## [1.0.5] - 2026-07-08
### Changed
- Removed the "View on Map" button from the Activity Log to streamline the UI.

## [1.0.4] - 2026-07-08
### Added
- Shifted the "Check for updates" button to the standard WordPress Plugin Row Meta (right side).

## [1.0.0] - Initial Release
### Added
- Core Fraud Prevention Engine.
- Custom Database Tables (`os_fraud_logs`, `os_rules`).
- Admin Dashboard with React-like tabs using vanilla JS.
- Automated GitHub release updater.
