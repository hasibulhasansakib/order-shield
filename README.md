# Order Shield - WooCommerce Fraud Prevention

<div align="center">
  <p>A production-ready, highly scalable fraud prevention and order protection system for WooCommerce.</p>
</div>

## 🛡️ Why Order Shield?

Fake orders, spam checkouts, and abusive customers can ruin a WooCommerce store. **Order Shield** provides an enterprise-grade defense mechanism to protect your store natively—without relying on expensive third-party SaaS platforms. This is the ultimate WooCommerce fraud prevention and security plugin.

Order Shield intercepts checkouts in real-time, validates the user's footprint (IP, Phone, Email), enforces strict daily limits, and logs detailed geographical data for every transaction. It's built for store owners who want complete control over their WooCommerce security.

## ✨ Features

- **Automated Fraud Prevention:** Stop fake orders instantly with intelligent rule engines.
- **Cart Snapshot Tracking:** Captures exactly what products a spammer tried to order, viewable in a beautiful premium UI modal.
- **Daily Order Limits:** Prevent abuse by capping the number of orders a user can place within 24 hours.
- **Advanced Blocklist:** Permanently block malicious IP addresses, emails, and phone numbers.
- **Geographic IP Tracking:** Automatically track the City, Country, ISP, and coordinates of every checkout attempt.
- **Log Cleanup Tool:** Built-in database management to easily delete old logs and keep your server lightning fast.
- **Automatic Failsafes:** Safely suspends itself if WooCommerce is deactivated, preventing critical errors.
- **HPOS Compatible:** 100% compatible with WooCommerce High-Performance Order Storage.
- **Blazing Fast:** Built with custom database tables to ensure zero impact on your checkout speed.
