# Security Policy

## Supported Versions

Currently, the latest version of Order Shield is fully supported for security updates.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability within Order Shield, please do NOT create a public GitHub issue.

Instead, please send an e-mail to **Hasibul Hasan Sakib** via the contact form on [hasibulhasansakib.com](https://hasibulhasansakib.com).
All security vulnerabilities will be promptly addressed.
