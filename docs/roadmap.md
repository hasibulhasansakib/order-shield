# Roadmap

Order Shield is continuously evolving. Here are the features planned for future releases:

### Short-Term
- [ ] Export/Import blocklist rules via CSV.
- [ ] Bulk actions for the Activity Log.

### Mid-Term (Pro Version Features)
- [ ] Centralized Blocklist (Sync blocks across multiple WooCommerce websites).
- [ ] ReCaptcha V3 Integration for checkout.
- [ ] Advanced Analytics Charting.

### Long-Term
- [ ] Machine Learning integration for predictive fraud scoring.
