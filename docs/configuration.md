# Configuration & Settings

Order Shield is designed to work out of the box, but you can configure it via **WooCommerce -> Order Shield -> Settings**.

## General Settings
- **Max Orders Per Day:** Set the maximum number of successful orders a single customer (IP/Phone) can place in 24 hours. Default is `3`.
- **Fake Phone Detection:** Automatically block orders with clearly fake phone numbers (e.g., `1234567890`, `00000000000`).

## Custom Messages
You can customize the error messages shown to customers on the checkout page:
- **Block Message:** The message shown to permanently blacklisted users.
- **Daily Limit Message:** The message shown to users who exceed their daily order quota.
