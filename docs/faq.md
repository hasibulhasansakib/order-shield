# Frequently Asked Questions (FAQ)

### How does Order Shield block fake WooCommerce orders?
Order Shield cross-references every checkout attempt with a high-performance Rules Engine. It checks the customer's IP address, phone number, and email against a global blocklist, and monitors daily order frequency to prevent spam attacks.

### Is Order Shield compatible with WooCommerce HPOS?
Yes! Order Shield is 100% compatible with WooCommerce High-Performance Order Storage (HPOS) and does not rely on legacy post tables.

### Does this slow down the checkout process?
No. Order Shield is designed for enterprise scalability. By using custom database tables and ultra-fast queries, the fraud check takes less than 5 milliseconds.
