# Contributing to Order Shield

First off, thank you for considering contributing to Order Shield! It's people like you that make Order Shield such a great tool for WooCommerce store owners.

## Development Setup

1. Fork the repo and clone it locally.
2. Run `composer install` to install PHP dependencies.
3. Activate the plugin in your local WordPress environment.

## Submitting Pull Requests

1. Create a new branch from `main`.
2. Ensure any new features are fully documented in the `/docs` directory.
3. Submit a pull request with a detailed description of your changes.
